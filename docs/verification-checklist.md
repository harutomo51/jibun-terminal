# Verification Checklist

Windows 11上で `npm run dev` を実行し、アプリ内のPowerShellプロンプトで確認します。

- [x] `dir` が動く
- [x] `Get-ChildItem` が動く
- [x] `cd` が動く
- [x] `git --version` が動く
- [x] `npm --version` が動く
- [x] 日本語入力ができる
- [x] コピー・貼り付けができる
- [x] ウィンドウリサイズで表示が崩れない
- [x] 背景プリセット切替ができる
- [x] 任意のアプリ背景色を選択できる
- [x] 任意のターミナル背景色を選択できる
- [x] ターミナル透明度をスライダーで連続調整できる
- [x] サイドパネルに現在のディレクトリ内のフォルダとファイルが表示される
- [x] サイドパネルの更新ボタンでファイル一覧を再読み込みできる
- [x] ファイルツリーでMarkdownファイルをクリックすると別ウィンドウでレンダリング表示される
- [x] ファイルツリーでHTMLファイルをクリックすると別ウィンドウでsandbox表示される
- [x] ファイルツリーでソースファイルをクリックすると別ウィンドウでコード表示される
- [x] `Ctrl+C` が効く
- [x] 対話CLIが起動できる
- [x] 終了時にPTYプロセスが残らない

## Notes

- PowerShell 7がある環境では `pwsh.exe` が優先されます。
- PowerShell 7がない環境では `powershell.exe` を使います。
- どちらも見つからない場合は画面上にエラーが表示されます。
- ファイルツリーは閲覧専用です。
